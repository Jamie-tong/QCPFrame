开始使用：

1. git init
2. git pull https://github.com/Jamie-tong/QCPFrame.git 或者 git pull https://gitee.com/JamieT/QCPFrame.git，后者为码云地址，速度快一点。
3. 打开 src/QCPF_HostView.pro
4. 项目设置->Shadow build去掉勾选
5. 编译运行。
6. 愉快的玩积木吧。
